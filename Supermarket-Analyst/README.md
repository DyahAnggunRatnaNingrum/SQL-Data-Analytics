Project ini bertujuan untuk menganalisis performa penjualan supermarket berdasarkan berbagai aspek seperti produk, cabang, tipe pelanggan, gender, metode pembayaran, dan tren waktu.
Analisis dilakukan menggunakan Google BigQuery, karena platform ini mendukung query cepat, integrasi mudah, serta cocok untuk dataset berukuran besar.

Dataset mencakup informasi berikut:
  1. Invoice ID
  2. Branch & City
  3. Customer type & Gender
  4. Product line
  5. Unit cost & Quantity
  6. Revenue, COGS, Gross Income, GM %
  7. Payment method
  8. Customer rating
